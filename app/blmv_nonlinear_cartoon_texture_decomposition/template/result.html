The texture image is obtained as the difference between original and cartoon images. The range of differences[-20,20] is linearly transformed to [0,255] for visualization purposes. Differences outside this range are saturated to 0 and 255 respectively. Details

The displayed error is:

  • 0 if the error is under -20
  • 128 + 128/20 * error if the error is between -20 and 20
  • 255 if the error is over 20
